This is the code for the Kaggle competition on predicitng the taxi fare given the pickup and dropoff locations.
https://www.kaggle.com/c/new-york-city-taxi-fare-prediction
The datasets for this competition are publicly available for download.
https://www.kaggle.com/c/new-york-city-taxi-fare-prediction/data
There is a much smaller version of the training file('train_small.csv') uploaded here just to test code.
